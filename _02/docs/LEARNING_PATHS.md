# Learning Paths (Canonical)

## Global baseline
- Required: Python 3.10-3.12
- Optional: Python 3.13 preview ideas as non-required context

## Path 1: Beginner

### Goal
Build confident fundamentals in I/O, functions, strings, debugging, and basic Git workflow.

### Sequence
1. `01_git_github_guide.md`
2. `02_vscode_setup_guide.md`
3. `03_input_output.py`
4. `04_functions.py`
5. `05_strings.py`
6. `06_debugging.py`
7. `07_practice_tasks.py`

### Expected outcomes
- Understand `input()`, `print()`, type conversion, basic formatting
- Write and call functions, use parameters and return values
- Apply core string methods for common text tasks
- Use debugger for step-by-step code inspection
- Complete beginner practice tasks end-to-end

## Path 2: Advanced DS/DE

### Goal
Apply Python fundamentals in practical DS/DE-like scenarios with stronger engineering hygiene.

### Sequence
1. `03_modern_input_output.py`
2. `04_modern_functions.py`
3. `05_modern_strings.py`
4. `06_debugging.py`
5. `07_practice_ds_tasks.py`

### Expected outcomes
- Validate and transform structured/unstructured input safely
- Use composable functions and reusable utilities for data workflows
- Apply robust string processing and regex-based extraction/cleaning
- Solve realistic task blocks (cleaning, aggregation, quality checks)
- Explain tradeoffs between simple and production-minded implementations

## Notebook alignment
- Primary teaching notebook for both paths: `fundamentals_notebook.ipynb`
- `notebook.ipynb` is legacy/optional and must not override canonical path order
