# Documentation Architecture — Module 6

## Canonical ownership
- Entry: `README.md`
- Learning order: `docs/LEARNING_PATHS.md`
- Timing: `docs/TEACHING_FLOW.md`
- File map: `docs/CONTENT_MAP.md`
- Legacy index: `docs/legacy/README.md`

## DRY rules
1. `START_HERE.md` лишається коротким маршрутизатором.
2. При зміні порядку тем оновлюється `LEARNING_PATHS`.
3. При зміні таймінгу оновлюється `TEACHING_FLOW`.
4. Legacy summary не використовуються як навчальні entry docs.

## Notebook policy
- Primary notebook: `Module_6_Complete_Course.ipynb`
