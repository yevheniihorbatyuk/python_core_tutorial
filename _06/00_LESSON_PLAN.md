# Модуль 6: Функціональне програмування та ООП
# Оновлений канонічний план заняття

## 1. Загальна інформація
- Тривалість: 4-5 годин
- Формат: theory + live coding + practice
- Рівень: intermediate
- Runtime: Python 3.10+

## 2. Принцип проведення
- Primary route обирається на старті (beginner або advanced).
- Root demo-файли (`01_*`, `02_*`) використовуються як спільний каркас.
- Деталізація треків — у `docs/LEARNING_PATHS.md`.

## 3. Канонічні джерела
- `README.md`
- `docs/CONTENT_MAP.md`
- `docs/LEARNING_PATHS.md`
- `docs/TEACHING_FLOW.md`
- `docs/ARCHITECTURE.md`

## 4. Таймінг заняття
### Block A. Setup and Context (10-15 хв)
- Цілі, scope, track selection

### Block B. Functional Programming (70-90 хв)
- Demo: `01_functional_programming.py`
- Track-specific приклади з `beginner_edition/` або `advanced_edition/`

### Block C. OOP Core (80-100 хв)
- Demo: `02_oop_classes.py`
- Розбір inheritance/composition/MRO

### Block D. Track Practice (45-60 хв)
- Beginner: `beginner_edition/01..05_*`
- Advanced: `advanced_edition/01_big_data_analytics.py` + конфіг/розширення

### Block E. Wrap-up and Q&A (10-15 хв)
- Підсумок і місток до серіалізації/системних тем

## 5. Expected outcomes
- Розуміння FP і OOP як практичних інструментів
- Вміння обирати підхід під задачу
- Виконана практика по активному треку
