# Teaching Flow (Canonical) — Module 4

```mermaid
flowchart TD
  A[Setup and Context 10-15m] --> B[Datetime + Math/Random 45-60m]
  B --> C[Regex + Files 45-60m]
  C --> D[Modules + Practice 40-55m]
  D --> E[Q&A and Wrap-up 10-15m]

  B --> B1[Beginner or Advanced/Pro]
  C --> C1[Beginner or Advanced/Pro]
  D --> D1[Track-specific practice]
```

## Правила
1. Один primary route на сесію.
2. Не дублювати повний material другого треку в межах одного таймслоту.
3. Практика має завершуватися runnable результатом.
