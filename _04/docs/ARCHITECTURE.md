# Documentation Architecture — Module 4

## Canonical ownership
- Entry: `README.md`
- Routes: `docs/LEARNING_PATHS.md`
- Timing: `docs/TEACHING_FLOW.md`
- Structure map: `docs/CONTENT_MAP.md`
- Legacy index: `docs/legacy/README.md`

## DRY rules
1. Не дублювати повні маршрути у `START_HERE.md`.
2. Зміна порядку тем: спочатку `docs/LEARNING_PATHS.md`.
3. Зміна таймінгу: спочатку `docs/TEACHING_FLOW.md`.
4. Статусні/історичні документи не тримати в root-навігації.
